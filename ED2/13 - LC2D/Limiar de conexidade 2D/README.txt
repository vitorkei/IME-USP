Comandos de compilação:
  javac LC2D.java

Comando de execução:
  java LC2D d T (S)
  > d é a distância máxima que dois pontos diferentes P_i e P_j podem estar um do outro para que sejam considerados conexos
  > T é o número de vezes que o quadrado Q de lados 1 será preenchido aleatóriamente
  > S é um argumento opcional. Determina a seed para gerar os pontos aleatórios
